# Food Economics Exam Trainer

Interactive study app for a Food Economics written exam.

## Features

- Past exam questions cleaned for grammar while preserving meaning.
- Generated two-option questions based on Maria Sassi's *Understanding Food Insecurity*.
- Real exam simulation with 11 questions, 3 points each, and pass threshold at 18/33.
- Optional explanation button for each question.
- Local progress tracking in the browser with accuracy and weak-topic detection.
- Mobile-friendly static app: no build step required.

## Use

Open `index.html` in a browser.
